# Design System Documentation

This document outlines the core aesthetic and functional principles of our design system.

## Theme Overview

The current design theme is configured for a **light** color mode, aiming for a **vibrant** feel.

## Color Palette

The color palette is built around the following key colors:

*   **Primary Color:** `#0082fd` (A bright, energetic blue, ideal for primary actions and branding.)
*   **Secondary Color:** `#5071d5` (A softer, slightly desaturated blue, providing support for less prominent elements.)
*   **Tertiary Color:** `#a459b5` (A warm, inviting purple, used for accents and highlights.)
*   **Neutral Color:** `#6a769e` (A cool gray, serving as a versatile base for backgrounds and text.)

## Typography

Our typography system prioritizes clarity and modern aesthetics:

*   **Headline Font:** `Inter` (Used for all major headings and titles, ensuring strong visual hierarchy.)
*   **Body Font:** `Inter` (Applied to all body text, promoting readability across various content types.)
*   **Label Font:** `Inter` (Utilized for form labels, buttons, and other interactive text elements.)

## Shape and Form

*   **Roundedness:** The UI elements feature a **moderate** roundedness (`2`), striking a balance between modern softness and crisp definition.

## Spacing

*   **Spacing:** A **normal** level of spacing (`2`) is applied throughout the UI, providing a balanced visual density that is neither too cramped nor excessively spacious.
