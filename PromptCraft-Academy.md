# PromptCraft Academy

**Tagline:** *Learn to talk to AI. Build anything.*

---

## Core Learning Tracks

### 1. Story Studio
- Kids write a prompt → AI generates an animated cartoon story
- They control characters, setting, plot twists via prompts
- Each story becomes a shareable mini-movie
- Teaches: descriptive prompting, storytelling structure

### 2. Web Builder Jr.
- Kids describe a webpage in plain English
- AI builds it live — they see the code AND the result
- "Make me a page about dinosaurs with a blue background and buttons"
- Teaches: structured prompts, cause & effect, basic HTML concepts

### 3. Game Maker
- Describe a simple game → AI generates it
- Flappy bird clone, quiz game, catch-the-star
- Kids iterate: "make it faster" / "add more enemies"
- Teaches: iterative prompting, refinement

### 4. Art Factory
- Text-to-image prompting
- Kids learn how adjectives, styles, moods change the image
- "A cartoon dragon in a rainforest at sunset in watercolour style"
- Teaches: specificity, descriptive language

### 5. Music Maker
- Describe a song/beat → AI generates it
- "A happy song for a birthday with drums and piano"
- Teaches: tone, mood, instrumentation prompting

### 6. Code Explainer
- AI explains real code in kids' language
- Kids ask questions about it using prompts
- Then modify it with plain English
- Teaches: understanding before building

## Learning Engine

### Prompt Score System
Rates every prompt on:
- **Clarity** — is it specific enough?
- **Creativity** — is it original?
- **Context** — did they give enough info?
- **Result** — did it work?

AI gives friendly feedback: *"Great start! Try adding what colour you want the dragon to be"*

### Prompt Library
- Kids save their best prompts
- Browse other kids' prompts for inspiration
- "Remix" other prompts and improve them

### Before & After
- Show a bad prompt vs improved prompt side by side
- Kids see the difference in output quality
- Core lesson: **better input = better output**
